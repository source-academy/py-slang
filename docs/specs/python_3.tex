\section{Changes}

Compared to Python \S 2, Python \S 3 has the following changes:
\begin{itemize}
  \item variable assignment
  \item literal list expressions
  \item list assignment
  \item list access expressions
  \item \textbf{\texttt{while}} loops
  \item \textbf{\texttt{for}} loops
  \item \textbf{\texttt{break}} and \textbf{\texttt{continue}}
  \item function
    \lstinline{is_list} is available, with its obvious meaning
  \item operators \texttt{==} and \texttt{!=} can be applied to
    arguments of any type
\end{itemize}
