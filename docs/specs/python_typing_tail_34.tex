
The operators
    \texttt{\textbf{==}},
    \texttt{\textbf{!=}},
    \texttt{\textbf{is}},
    \texttt{\textbf{>}},
    \texttt{\textbf{>=}},
    \texttt{\textbf{<}},
    \texttt{\textbf{<=}}
    follow the 
    \href{https://docs.python.org/3.14}{\color{DarkBlue}Python 3.14 Documentation}
    except that the operators \texttt{\textbf{is}} and
    \texttt{\textbf{is not}} must not be applied to
    numbers, strings, and boolean values (recommended in Python but not enforced
    by Python 3.14).

    As in Python, \texttt{\textbf{True}} and \texttt{\textbf{False}} are
    represented by the integers 1 and 0, respectively, for the purpose of
    any comparisons.
